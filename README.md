Projeto de landing page construido com 3 blocos principais, sendo eles, Header, Main e Footer.

O Header, contem apenas a logomarca do projeto, centralizado e com a margem aplicada ao topo.

O Main é dividido em 2 partes, 

Sendo a primeira nomeada de Principal onde contem Titulo e Subtitulo, alem da imagem

A segunda contem o texto completo da materia e foi nomeada como Content

As fontes usadas no mair são, Inter-black e Inter-Regular
